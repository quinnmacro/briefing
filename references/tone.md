# 公文体写作规范 — 参阅材料版

## 总体原则

参阅材料属于"内部参阅、高层决策支持"性质，文风介于公文与研究报告之间：
- **精确**：每个数据点必须有出处和时点
- **简洁**：一句话讲一个事实，不铺陈、不抒情
- **结构化**：bold lead-in + 数据支撑的标准段落格式
- **中性**：陈述事实和趋势，不做主观判断（除非标注"市场预期"）

## 段落格式

### 要点段落（最常用）
```json
{ "type": "body_para", "runs": [
    { "text": "1. 香港增长动能强劲。", "bold": true },
    { "text": "2026年一季度本地生产总值同比实质增长5.9%..." }
  ]
}
```
- bold lead-in：1-2句话概括，加粗
- body：数据展开，不加粗
- 首行缩进560 DXA（≈2字符）

### 叙述段落
```json
{ "type": "body_para", "text": "一季度本地生产总值同比实质上升5.9%..." }
```
- 首行缩进，两端对齐

### 列表段落
```json
{ "type": "flush_para", "text": "• 私人消费开支同比+5.0%..." }
```
- 不缩进，bullet开头

### 编号要点
```json
{ "type": "num_item", "n": "1", "text": "要点内容" }
```
- 中文编号：1、2、3...（不是1. 2. 3.）

## 数字表述

- 百分比："+5.9%"（正号）、"-0.3%"（负号）、"5.0%"（零附近可省略正号）
- 金额：1,104亿港元（千位逗号）、295亿元（无逗号当<1万）
- 增幅：同比+719%（极端增幅照实写，不省略）
- 区间：25,000—27,000（中文连接号——）
- 时点：2026年5月26日（全写）、4月（月度简写）
- 对比："为去年同期近6倍"（倍数）、"较上月扩大0.4个百分点"（百分点）

## 禁忌

- ❌ 不用"据悉"、"据了解"等模糊来源
- ❌ 不用"值得关注"、"令人瞩目"等主观评价
- ❌ 不用英文缩写替代中文（写"一季度"而非"Q1"，但"IPO"、"CPI"、"GDP"等通用缩写可用）
- ❌ 不用比喻、排比、感叹号
- ✅ 用"数据显示"、"据统计"、"根据xxx"明确出处
- ✅ 用"创近五年新高"、"维持低位"等可验证描述

## 章节标题格式

- 一级：一、xxx（中文数字+顿号）
- 二级：（一）xxx（中文数字+括号）
- 三级：表一 xxx（表格标题）
